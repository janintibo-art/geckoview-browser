# GeckoBrowser

Navigateur Android minimal basé sur GeckoView (moteur de Firefox).

## Compilation automatique
À chaque push sur `main`, GitHub Actions compile l'APK.
Récupérez-le dans l'onglet **Actions** → dernier run → artefact **GeckoBrowser-debug**.

Vous pouvez aussi lancer le build manuellement : Actions → Build APK → *Run workflow*.
