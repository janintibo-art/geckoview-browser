Mes flux

Les nouveautes des sites que vous suivez, meme ceux qui ne proposent aucun flux.

Pour creer un flux : ouvrez la page qui liste les articles, puis Menu › Page › Creer un flux. Vous designez un article, le navigateur reconnait les autres.

Les entrees presentes a la creation sont marquees comme deja vues : seules les suivantes apparaitront en nouveaute. Le relevé a lieu pendant que le navigateur tourne, toutes les trois heures par flux par defaut.
